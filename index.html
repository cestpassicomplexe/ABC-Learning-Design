Utilisation de l'outil d'aide à la conception de parcours scénarisé

Le modèle ABC Learning Design

Cet outil est basé sur la taxonomie ABC Learning Design développée par Diana Laurillard (UCL). Il vous aide à structurer votre cours en vous concentrant sur les activités d'apprentissage réelles de vos étudiants.

Conception Manuelle

La méthode se base sur 6 types d'apprentissage (Acquisition, Enquête, Discussion, Collaboration, Pratique, Production). Cliquez sur "Ajouter une activité" dans la ligne du temps pour organiser votre séquence de cours, puis personnalisez votre séquence à l'aide du tableau de scénarisation en y renseignant les consignes, outils, durées, etc.
Le conseil pédagogique : Variez les types d'apprentissage pour éviter la monotonie et engagez les apprenants !

Conception avec l'Assistant IA
Génération Intégrée (via Clé API)

Dans le menu "Assistant IA > Configuration IA", vous pouvez pour le moment renseigner uniquement une clé API Gemini. Une fois configurée, allez dans "Générer une séquence". Précisez vos objectifs, le niveau, la durée, et les plateformes privilégiées (Moodle, Teams...). L'IA va automatiquement générer et prévisualiser une séquence équilibrée que vous pourrez insérer d'un simple clic au tableau.

Génération Externe (Copié-collé)

Si vous n'avez pas de clé API ou utilisez une autre IA, allez dans "Assistant IA > Générer un Prompt Manuel". Ce menu va préparer pour vous un texte ultra-précis ("Prompt") basé sur vos contraintes.
Comment faire ? Copiez ce texte généré, collez-le dans ChatGPT, Claude ou l'IA de votre choix. L'IA va vous générer en réponse un bloc de code au format `.json`. Copiez ce JSON, revenez sur cet outil, et utilisez "Charger > Depuis le texte / presse-papiers" pour importer instantanément le scénario généré.

Sauvegarde et Exports

N'oubliez pas d'enregistrer votre travail en cliquant sur Sauvegarder. Vous pourrez aussi exporter votre tableau sous différents formats (Word, Excel, Markdown, ou HTML).

Attention : L'import (Charger) ne fonctionne qu'au format JSON original. Il n'est pas possible de réintégrer un scénario que vous auriez exporté puis modifié au format Excel, Word ou HTML.

Choisir un type d'apprentissage

Sélectionnez le type d'activité que vous souhaitez ajouter à votre séquence :

Acquisition

L’apprenant écoute un cours magistral, un podcast ; lit un livre ; regarde des démonstrations/vidéos.

Collaboration

Activités en équipe. Participer activement à la construction collective de la connaissance.

Discussion

Apprendre en formulant ses idées et questions, en écoutant et répondant aux arguments de tiers.

Enquête

Explorer, comparer et critiquer les ressources qui renvoient aux concepts et idées enseignés.

Pratique

S’entraîner et utiliser la rétroaction (enseignant, pairs ou activité elle-même) pour progresser.

Production

Consolider ses acquis par une production sur des supports variés (bilans, rapports, créations).
